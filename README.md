# 🇺🇸 English

## 🌟 Introduction

**Banner CLI** is a powerful Node.js-based tool that allows you to generate stunning ASCII banners with advanced styling.

It goes beyond simple ASCII generation by supporting:

* 🎨 Gradient, Rainbow, Neon color systems
* 🧱 Layout effects (Box, Double Border, Rounded, Shadow)
* ⚡ Glitch text effects
* 📦 Export to multiple programming languages (JavaScript, Python, C#, etc.)

Perfect for:

* CLI tools
* Discord bots
* Terminal branding
* Developer portfolios

---

## 🛠 Installation

```bash
npm install
```

> ⚠ Requires Node.js >= 16

---

## 💻 Usage (Non-Interactive Mode)
### 🔹 Normal Command
```bash
node src/index.js
```

Run everything instantly via CLI arguments (no prompts required).

### 🔹 Basic Command

```bash
node src/index.js "Your Text" --style <preset> --export --exportLang <language>
```

### Run File
```bash
node banner.js
```

---

### 🔹 Examples

#### Hacker style + export Python

```bash
node src/index.js "JS-ARMOR" --style hacker --export --exportLang python --exportVar my_banner
```

#### Neon style + center align

```bash
node src/index.js "WELCOME" --style neon --align center
```

#### Retro style + export JavaScript

```bash
node src/index.js "NODEJS" --style retro --export --exportLang javascript
```

## 📂 Project Structure

```bash
src/
├── index.js     # CLI entry point, argument parser & main flow
├── prompt.js    # Interactive CLI (optional mode)
├── preview.js   # Render pipeline (font → effect → color → align)
├── font.js      # Figlet integration & font manager
├── color.js     # ANSI colors, gradient, neon logic
├── effect.js    # Box, shadow, glitch, layout effects
├── codegen.js   # Multi-language export engine (10+ languages)
├── file.js      # File writing handler
├── preset.js    # Preset definitions
└── i18n.js      # Multi-language support (EN / VI)
```

### Example outputs

**`banner.js`** (from `node bin/banner-cli.js "Hi" --style neon --export --export-lang javascript --export-var greeting`)

```js
// Generated by banner-cli
const greeting = "\x1b[1m\x1b[38;2;255;61;241m \x1b[39m...    __  ___ ...\x1b[22m";

console.log(greeting);

export default greeting;
```

**`banner.py`**

```python
# Generated by banner-cli
greeting = """\x1b[1m\x1b[38;2;255;61;241m ...    __  ___ ...\x1b[22m"""

if __name__ == "__main__":
    print(greeting)
```

**`banner.go`**

```go
// Generated by banner-cli
package main

import "fmt"

const greeting = "\x1b[1m\x1b[38;2;255;61;241m ...    __  ___ ...\x1b[22m"

func main() {
    fmt.Println(greeting)
}
```

**`banner.sh`**

```bash
#!/usr/bin/env bash
# Generated by banner-cli
GREETING=$'\e[1m\e[38;2;255;61;241m ...    __  ___ ...\e[22m'

printf '%s\n' "$GREETING"
```
